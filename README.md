Me Make It cURL Module
======================

This module provides a wrapper for cURL HTTP requests.

The various HTTP methods map to the following class methods:
>delete($url, $parms = NULL)
>get($url, $parms = NULL)
>head($url, $parms = NULL)
>post($url, $parms = NULL)
>put($url, $parms = NULL)
